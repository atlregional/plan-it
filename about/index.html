About

This site was built in response to a few "trends" that we think have lasting impacts on the way we manage data and software.

Trend 1: Open Data

Government is for the people, so why shouldn't the data it produces be likewise?

Trend 2: Open Source

Since its genesis only a few short years ago, open source software has been transforming the software development industry. Open source software makes all the more sense in the world of government when there are hundreds of agencies around the country (the world, even) doing very similar things.

Trend 3: GitHub

Hosting source code on GitHub is old hat. However, there are some very interesting use cases developing for hosting data on GitHub, especially for government. We think TIP data and the process that surrounds it is one of those use cases.
