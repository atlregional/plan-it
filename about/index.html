About

This site was built in response to a few "trends" that we think have lasting impacts on the way we manage data and software.

Trend 1: Open Data

Government is for the people, so why shouldn't the data it produces be likewise? In May 2013, President Obama agreed and issued an executive order requiring all federal agencies to create and implement open data policies. Check out data.gov for a directory of national datasets.

Trend 2: Open Source

Since its genesis only a few short years ago, open source software has been transforming the software development industry. Open source software makes even more sense in government where there are hundreds of agencies around the country doing similar things.

Trend 3: GitHub

Hosting source code on GitHub is old hat. However, there are some very interesting use cases developing for storing data on GitHub, especially for government. Add in GitHub pages and other features, and your toolbox becomes truly extensive.
