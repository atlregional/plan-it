Pivot Table

Pivot tables are great way to summarize large datasets.

The data for the table below are from the most recent TIP. It is designed to have many fields that have at least some significant relationships between one another. Check out the README for more information. Also feel free to run the spec (found in the link up top), and if you spot any problems let us know so we can fix them.

To begin:

  • Select the fields you would like to display from the Label Fields/Summary Fields drop down.
  • Then filter your data by selecting from the filters drop down.
  • Ta-da, you have yourself a custom report!
